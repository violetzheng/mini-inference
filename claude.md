# Project overview
A simple, lightweight LLM inference engine in cpp for learning purposes. Existing similar projects include ilama.cpp, but I want this project to optimize for readability, clarity, simple architecture, but accessibility rather than production performance.

# Code style
Clear, well-structured code. Concise comments as explanation before complex or ambiguous code only. Use snake style variable names, lower case file namings, camel case for class names. Write unit tests for main features. 

# Architecure
Separate main modules into submodules (e.g. tokenizer, tensor, model loader etc). 

# Development
